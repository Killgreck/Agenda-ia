Instructions for GitHub Upload:

  1. Go to https://github.com/Killgreck/Agenda-ia
  2. Click on the "Upload files" button
  3. Drag and drop all files from the project directory (excluding node_modules)
  4. Add a commit message like "Initial upload of Agenda-IA project"
  5. Click "Commit changes"

What's included in your project:

  • README.md - Project documentation
  • LICENSE - MIT License
  • .gitignore - Git configuration
  • All source code files for both frontend and backend
  • Configuration files (package.json, etc.)

Note: When uploading to GitHub, exclude the node_modules directory as it's very large and not needed in the repository. Users can run npm install after cloning.

Alternative Method: Using Git Command Line

If you prefer using the Git command line, you can clone the empty repository and push your code:

git clone https://github.com/Killgreck/Agenda-ia.git
# Copy all files (except node_modules) to the cloned directory
cd Agenda-ia
git add .
git commit -m "Initial upload of Agenda-IA project"
git push

You'll need to authenticate with your GitHub credentials when pushing.
